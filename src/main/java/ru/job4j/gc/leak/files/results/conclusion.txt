Для устранения утечек памяти и оптимизации кода были проведены следующие мероприятия:
Исключена привязка полей типа List и Map к static; Произведена проверка на использование try-with-resources;
Переопределены методы equals и hashCode в модели Post для последующего предотвращения появления дубликатов в HashMap;
Исключены переопределенные методы finalize() для всех моделей.
