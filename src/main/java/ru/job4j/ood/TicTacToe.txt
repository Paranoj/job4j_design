Board - создает квадратное поле размера от 3х3 и более.

ValidateInput - проверяет, что пользователь ставит фигуру в пределах границ поля и на свободную клетку.

Input - считывает позицию для фигуры, которую задает пользователь. Пользователь, который ходит первый играет за "крестики".

Figures - описывает "крестик" и "нолик".

Rules - описывается условие победы, поражения и ничьи.

Rotation - описывает логику чередования ходов.

Output - вывод сообщения о результате партии.

Main - реализует общую логику работы приложения.
